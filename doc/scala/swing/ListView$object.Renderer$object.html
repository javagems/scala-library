def apply [A, B](f : (A) => B)(implicit renderer : Renderer[B]) : Renderer[A]
Returns a renderer for items of type A. The given function converts items of type A to items of type B for which a renderer is implicitly given. This allows chaining of renderers, e.g.: case class Person(name: String, email: String) val persons = List(Person("John", "j.doe@a.com"), Person("Mary", "m.jane@b.com")) new ListView(persons) { renderer = ListView.Renderer(_.name) }
